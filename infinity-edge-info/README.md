## 💻 How to Use

### Local Installation
1. Download/Clone the repository
2. Navigate to `infinity-edge-info` folder
3. Open `index.html` in your web browser
4. No server or dependencies required!

### Deploy to GitHub Pages
1. Push files to GitHub repository
2. Go to Settings → Pages
3. Select branch `main` and folder `/infinity-edge-info`
4. Your site will be live at `https://username.github.io/repository/infinity-edge-info/`
